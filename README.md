# Gustito a Salta - Sistema de Pedidos Web



## Descripción

Este proyecto es una página web optimizada para dispositivos móviles que permite a pequeños emprendimientos gastronómicos gestionar pedidos de empanadas y bebidas de manera simple y accesible. A través de una interfaz intuitiva, los clientes pueden seleccionar productos y enviar su pedido directamente a WhatsApp.

Este desarrollo fue realizado en el marco del **voluntariado autogestionado de la Práctica Profesional de Teclab**, con un enfoque **low-cost** para negocios que buscan una solución digital sin costos de mantenimiento elevados.

## Características Principales

- **Interfaz optimizada para móviles** 📱
- **Pedido rápido a través de WhatsApp** 📩
- **Gestión de productos y precios mediante Google Sheets** 📝
- **Modo mantenimiento configurable desde Google Sheets** ⚙️
- **Diseño simple y ligero sin necesidad de hosting avanzado** 🚀

## Tecnologías Utilizadas

- **HTML5** - Estructura semántica
- **CSS3** - Diseño responsivo
- **JavaScript (Vanilla JS)** - Interactividad y lógica del pedido
- **Google Sheets** - Backend para gestión de productos y estado del negocio

## Instalación y Uso

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/Marcktm/GustitoOnline.git
   ```
2. Abrir el archivo `index.html` en cualquier navegador.
3. Personalizar la hoja de Google Sheets para agregar o modificar productos y precios.
4. Ajustar el número de WhatsApp en el script (`script.js`).
5. Subir la web a un hosting gratuito o usar GitHub Pages para compartirla.

## Configuración de Google Sheets

La web se conecta a dos hojas de cálculo públicas:

1. **Estado del negocio** (habilitado/mantenimiento).
2. **Lista de productos con precios y stock disponible**.

Para modificar los productos, simplemente edita la hoja de cálculo vinculada. No es necesario tocar el código.
